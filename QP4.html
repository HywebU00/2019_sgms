  2. 會員依團體性質擇用適當之類別名稱，例如：個人會員、團體會員、預備會員（或準會員）、永久會員、贊助會員、榮譽會員（或名譽會員）、學生會員等。

第八條、

會員（會員代表）基於平等原則，均享有以下權利：

  1. 表決權。

第九條、

會員（會員代表）有以下之義務：

註：團體可自行增訂會員（會員代表）應遵守之義務。

第十條、

會員（會員代表）有違反法令、章程或不遵守會員大會決議時，得經理事會決議，予以下列處分：

  1. 警告。
  2. 停權。補充

前項決議如經理事會審議通過得免除其處分。

註：團體可自行增訂會員（會員代表）違反義務之處分。

第十一條、

會員(會員代表)有下列情事之一，經會員(會員代表)大會決議通過者，喪失其資格：

  1. 除名。
  2. 連續 年未繳納會費。
  3. 死亡。補充

如欲恢復其資格者，應提經會員(會員代表)大會通過。

註：團體可自行增訂會員（會員代表）喪失資格之條件。

第十二條、

會員（會員代表）得以書面敘明理由向本社團聲明退會。

第十三條、

本社團以會員(會員代表)大會為最高權力機構。
會員得推舉會員代表，再召開會員代表大會，行使會員大會職權。(本社團 會員代表)

第十四條、

會員（會員代表）大會之職權如下：

  1. 章程之訂定及變更。
  2. 理事或監事之選任及罷免。
  3. 年度之預算、工作計畫與決算及工作報告。
  4. 不動產之處分、設定負擔及購置。
  5. 會員（會員代表）之除名。
  6. 合併或分立。
  7. 解散。補充

(其他依章程規定應經會員（會員代表）大會決議事項。)
前項第一款、第四款、第六款及第七款之決議，應有全體會員（會員代表）過半數之出席，出席人數三分之二以上之同意。前項第二款、第三款、第五款之決議，應有會員（會員代表）過半數之出席，出席人數過半數之同意行之。

第十五條、

本社團置理事 人、監事 人，分別成立理事會、監事會。
選舉理事、監事時，依計票情形得同時選出候補理事 人、候補監事 人。
遇理事、監事出缺時，分別依序遞補之。

註：

  1. 理事及監事職稱得自定之。(理事職稱 、監事職稱 )
  2. 理事及監事名額應各為3人以上且為奇數。
  3. 候補理事、候補監事得不設置。
  4. 候補理事及候補監事職稱得自定之。(候補理事職稱 、候補監事職稱 )

  2. 議決理事及負責人之辭職。

第四章 會議

第二十二條、

會員（會員代表）大會應以集會方式(含視訊)為之，由負責人召集，每年至少召開1次，並應於15日前發出通知，但召開臨時會議，應於開會前1日通知。通知方式得以書面或其他方式為之。
臨時會員（會員代表）大會於理事會認為必要，或經會員（會員代表）十分之一以上之請求，或監事會函請召集時召開之。
會員（會員代表）需親自出席大會，且應過半數之出席，始得開會。
會員（會員代表）不能親自出席大會時，得以書面委託其他會員（會員代表）代理，每一會員（會員代表）以代理1人為限。
會員（會員代表）大會決議，應有會員（會員代表）過半數之出席，出席人數過半數同意行之。
會員（會員代表）大會不能依法召開時，應由理事互推1人或由會員（會員代表）過半數推薦1人召集之。

第二十三條、

理事會每 個月召開1次，並應於15日前發出通知，但召開臨時會議，應於開會前1日通知。通知方式得以書面或其他方式為之。

理事會由負責人召集。理事應出席（含視訊）理事會議，不得委託出席，且應過半數之出席，始得開會。理事會之決議，以理事過半數之出席，過半數或較多數同意行之。
理事會不能依法召開時，應由理事互推1人召集之。
理事連續二次無故缺席者，視同提出辭職。

第二十四條、

監事會每 個月召開1次，並應於15日前發出通知，但召開臨時會議，應於開會前1日通知。通知方式得以書面或其他方式為之。

監事會由監事會召集人召集。監事應出席（含視訊）監事會議，不得委託出席，且應過半數之出席，始得開會。監事會之決議，以監事過半數之出席，過半數或較多數同意行之
監事會不能依法召開時，應由監事互推1人召集之。
監事連續二次無故缺席者，視同提出辭職。

第二十五條、

理事會、監事會得於必要時召開臨時會議。經理事、監事各過半數之連署，得由負責人召開理事監事聯席會議。

第五章 經費及會計

第二十六條、

本社團經費來源如下：

  1. 入會費：
    1. 個人會員 : 新台幣 元，於入會時繳納。
    2. 團體會員 : 新台幣 元，於入會時繳納。
  2. 常年會費：
    1. 個人會員 : 新台幣 元。
    2. 團體會員 : 新台幣 元。
  3. 事業費。
  4. 會員捐款。
  5. 委託收益。
  6. 基金及其孳息。
  7. 其他收入。

註：入會費常年會費繳納標準及方式，請配合章程第7條會員種類，分別訂定應繳金額，並應於章程明定之。

第二十七條、

本社團之財產、孳息及各項所得，不得有分配盈餘行為。

第二十八條、

本社團會計年度採曆年制，自每年1月1日起至12月31日止。

註：經外交部認可之國際組織在我國設立之社會團體章程另有規定者，從其規定。

第二十九條、

本社團最遲於每年3月底前，由理事會編造當年度之工作計畫書、收支預算表，提經會員（會員代表）大會通過後公開之。

註：社團經費來源僅為入會費、常年會費且年度收入決算數或資產總額未達中央主管機關公告之一定金額者，得不公開之。

第三十條、

本社團最遲於每年3月底前，由理事會編造上年度之工作報告、會計報告，送監事會監察通過後，提會員大會通過後公開之。

註：社團經費來源僅為入會費、常年會費且年度收入決算數或資產總額未達中央主管機關公告之一定金額者，得不公開之。

第三十一條、

本社團之決算及工作報告，應於理事會議時提出審議，並由理事會送請監事會監察，監事會監察發現有不當情事者，應提出糾正意見，送請理事會處理，如理事會不為處理時，監事會得提報會員（會員代表）大會審議。

第三十二條、

本社團於解散後，其賸餘財產應歸屬該機關或團體所在地之地方自治團體，或政府主管機關指定之機關或團體。

本章程未規定事項，悉依有關法令規定辦理。

第三十四條、
